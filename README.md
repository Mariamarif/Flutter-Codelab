# Flutter Codelab

This repository contains my Flutter learning project based on the Google Flutter Codelab: Write your first Flutter app.
I extended the original app with additional features and UI improvements.

## About the App

The Namer App generates random word pairs and lets you interact with them in a fun and structured way.

## Features

✔ Generate random word pairs

✔ Like word pairs

✔ Dislike word pairs

✔ View Favorites on a separate page

✔ View Disliked words on a separate page

✔ Delete items from Favorites or Disliked lists

## How to Run the App

* Clone this repository:
git clone https://github.com/Mariamarif/Flutter-Codelab.git

* Navigate into the project:
cd Flutter-Codelab

* Get dependencies:
flutter pub get

* Run the app:
flutter run
